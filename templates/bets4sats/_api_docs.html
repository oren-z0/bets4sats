Bets4Sats lets you bid on a competition and pays out rewards to the winner/s.

This extension allows you to manage bets between friends on your competitions. Each user chooses an option to bet on as well as a lightning-address or a lnurlp (must be on another lnbits instance) to receive the price. When the admin of the competition sets the option that has won it will trigger those reward/s to be send.
Bets4Sats comes with a shareable ticket scanner, that can be used to look up tickets (i.e. in case of a question).

Competition States:
- INITIAL: No winning option selected yet
- COMPLETED_PAYING: Winning choice was selected. If cancelled refunds will be paid out.
- COMPLETED_PAID: All winners found.

Ticket States:
- INITIAL: Winning option was not selected yet.
- LOST: This Ticket has made a wrong bet.
- WON_UNPAID: This ticket is from a winner! Searching more..
- WON_PAYING: This option has won, start paying out the price..
- WON_PAYMENT_FAILED: Paymemt to winner failed.
- WON_PAID: Payment to winner successful.
- CANCELLED_UNPAID, CANCELLED_PAYING, CANCELLED_PAYMENT_FAILED, CANCELLED_PAID: Same as WON_UNPAID, WON_PAYING, WON_PAYMENT_FAILED, WON_PAID, but in case the competition was cancelled and the user deserves to get a refund instead of a reward.
